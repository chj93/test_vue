컴포넌트 사용시 주의할 점
    1. 템플릿 문자열 안에서 루트 요소는 하나여만 함

        -문제 상황
        <template id="ex1">
            <div>1</div>
            <div>2</div>
        </template>

        -해결 : root div 생성
        <template id="ex1">
            <div>
                <div>1</div>
                <div>2</div>
            </div>
        </template>

    2. html요소는 자식요소로 포함시킬 수 있는 요소들이 정해져 있으므로 구문분석 오류 발생('optinn is' 사용)
        - .vue확장자를 쓰는 single file component일 경우 제외

        <div> id="app">
            <select>
                <option is="option-component"></option>
                <option is="option-component"></option>
            </select>
        <div/>

    3. data option
        컴포넌트에서 데이터 옵션을 함수로 지정할 때 리턴값은 반드시 함수 내부에 선언된 객체여야
