Vue.js : component를 조합하여 작성

Component : 부모-자식 관계로 트리 구조 형셩. 부모가 자식을 포함하는 형태

        [부모] ----속성(Props)----> [자식]
        [부모] <-------Event------ [자식]
    EventHandler               EventTrigger

          - .sync 수식어 이용하여 양방향 속성 전달 가능하나 권장하지 x

    대부분의 instance option 사용 가능하나, data option 사용시 반드시 함수로 작성하고, 함수 내부에서 객체를 리턴하도록 작성
    (컴포넌트에서 데이터 옵션은 각 컴포넌트의 로컬 상태를 관리하기 위한 용도, 객체 값으로 작성할 수 없고 모두 동일한 참조형 값)

    선언방식
        ex1 | inline 선언
        Vue.component('ex1', {
            template : '<div>Ex1!!</div>'
            //inline template : 문자열을 사용해 템플릿 지정
        })

        ex2 | xml로 선언
        <template id="helloTemplate">
            <div>hello world!!!</div>
        </template>
        <script type="text/javascript">
            Vue.component('hello-component', {
                template : '#helloTemplate'
                //id 값으로 component 찾아가도록
            })
        </script>

        ex3 | script로 선언
        <script type="text/x-template" id="helloTemplate">
            <div>hello world!!!</div>
        <script type="text/javascript">
          Vue.component('hello-component', {
              template : '#helloTemplate'
          })
        </script>
